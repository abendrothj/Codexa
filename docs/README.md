# Codexa Documentation

## Overview

Codexa is a local-first AI dev knowledge vault that enables semantic search over your development documentation and code files. It uses ChromaDB for vector storage, SentenceTransformers for embeddings, and provides both a REST API and desktop GUI.

## Features

- **Semantic Search**: Find relevant code and documentation using natural language queries
- **Local-First**: All data stays on your machine
- **Encryption**: Optional AES-256 encryption for sensitive content
- **File Support**: Built-in parsers for Markdown (.md) and Python (.py) files
- **REST API**: FastAPI-based backend for flexible integration
- **Desktop GUI**: User-friendly PySide6 interface
